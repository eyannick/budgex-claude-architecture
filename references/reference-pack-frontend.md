# reference-pack-frontend.md

## Rôle du document

Ce pack fournit les références prioritaires à consulter pour une tâche frontend liée aux templates Twig, aux composants UI, aux tableaux, au responsive ou à l’accessibilité.

## Références prioritaires

- Bootstrap 5.3
- DataTables
- Twig docs
- WCAG 2.2
- WAI-ARIA Authoring Practices

## Quand le charger

- tâche liée à Twig, Bootstrap, DataTables ou structure HTML ;
- problème de responsive, lisibilité ou accessibilité ;
- besoin de vérifier un pattern d’interface ou un comportement frontend standard.

## Règles d’usage

- commencer par les templates, assets et composants déjà présents dans le projet ;
- consulter Bootstrap avant d’introduire une structure ou un composant non standard ;
- consulter WCAG / WAI-ARIA dès qu’un changement touche navigation, formulaires, tableaux, modales ou interactions clavier ;
- ne charger DataTables que si la tâche concerne réellement un tableau interactif ;
- ne citer que les références réellement utilisées.

## Hors périmètre de ce pack

Ce document n’est pas la source primaire pour :
- les règles de design système Budgex ;
- les décisions produit ;
- la logique métier backend ;
- les arbitrages UX transverses.
