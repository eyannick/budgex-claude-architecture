# reference-pack-seo.md

## Rôle du document

Ce pack fournit les références prioritaires à consulter pour une tâche SEO technique liée aux pages publiques, aux métadonnées, aux signaux d’indexation ou aux données structurées.

## Références prioritaires

- Google Search Central
- Open Graph
- schema.org

## Quand le charger

- tâche liée aux titres, metas, canonical, robots, sitemap ou Open Graph ;
- besoin de vérifier un signal SEO technique sur une page publique ;
- besoin d’évaluer l’intérêt ou la validité de données structurées.

## Règles d’usage

- charger ce pack uniquement pour le SEO technique public ;
- ne pas l’utiliser pour les zones privées, `/app/` ou `/admin/` ;
- consulter Google Search Central pour les règles d’indexation et de découverte ;
- consulter Open Graph pour les métadonnées de partage ;
- consulter `schema.org` uniquement si des données structurées sont réellement pertinentes ;
- ne citer que les références réellement utilisées.

## Hors périmètre de ce pack

Ce document n’est pas la source primaire pour :
- la logique métier ;
- la stratégie produit ;
- les décisions d’exposition des zones privées ;
- les règles de sécurité ou d’authentification.
